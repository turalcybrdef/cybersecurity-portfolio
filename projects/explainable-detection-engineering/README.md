## Project Overview
This project designs and implements an explainable, rule-based detection engineering system aligned with the MITRE ATT&CK framework. The system collects host-based telemetry from Windows and Linux endpoints, normalizes events into a unified structure, and applies ATT&CK-mapped detection logic with clear supporting evidence.

The focus of this project is transparency, repeatability, and investigative clarity rather than opaque anomaly-based detection.
